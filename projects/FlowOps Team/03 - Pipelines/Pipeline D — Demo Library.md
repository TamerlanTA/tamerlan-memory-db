# 🎬 Pipeline D — Demo Library

#flowops #pipeline #demos

> [!abstract] Goal
> Build a library of 10 short Loom demo videos showing real automation systems in action — one per niche/offer. Use these in proposals, LinkedIn outreach, cold emails, and after discovery calls to accelerate decision-making.

---

## 🎯 Why a Demo Library?

> [!tip] The Demo Principle
> Seeing is believing. A 2-minute Loom that shows a real workflow running is worth 10 pages of explanation. Demos collapse the trust gap faster than any other sales tool.

**Where demos get used:**
- Attached to Upwork proposals
- Linked in LinkedIn DMs
- Sent after cold email openers
- Shown live on discovery calls
- Sent as follow-up after calls

## 🎥 Loom Format

**Ideal length:** 2–4 minutes (never more than 5)

**Structure for each demo:**

```
[0:00–0:20] Hook
"This is what happens when a new lead comes in for [niche] — watch."

[0:20–1:30] The Workflow in Action
Show the trigger → show each step → show the output
No explanation while it's running — let it speak for itself

[1:30–2:30] Break Down What Just Happened
"So what you just saw was: [plain English summary]"
"This took [X hours/week] for [client type] — now it's zero."

[2:30–3:00] Close
"If your business does [X], this is probably relevant to you.
Drop me a message if you want to see this built for your setup."
```

---

## 🎬 Demo Script Template

```
HOOK (10 sec):
"I'm going to show you something that took [pain task] from 
[X hours] to completely automatic. Watch."

DEMO RUN (60–90 sec):
[Screen record the live workflow — real data, real tools]
Talk minimally during the run — let the visuals carry it

WHAT YOU SAW (30 sec):
"What just happened: [trigger] → [action 1] → [action 2] → [result]
Every time. Automatically. No human needed."

SOCIAL PROOF (15 sec):
"We've built this for [niche] businesses and typically save 
[X hrs/week] or recover [$ amount] in lost leads monthly."

CTA (15 sec):
"If you want to see what this looks like for your specific 
setup, reply to this message or book a free 20-min audit call."
```

## 📁 Demo Organization

**Storage:** Notion database OR Google Drive folder with:
- Demo video link (Loom)
- Offer category
- Target niche
- Best use case (Upwork / LinkedIn / cold email / post-call)
- Personalization notes

**Naming convention:**
`[OFFER]-[NICHE]-[VERSION].loom`
e.g., `speed-to-lead-real-estate-v1.loom`

---

## ✅ Demo Build Checklist

- [ ] Use realistic but fictional/anonymized data
- [ ] Show the actual tools (Make.com, Airtable, Slack, etc.) — this builds credibility
- [ ] Thumbnail is a compelling frame (not a blank screen)
- [ ] No dead air or loading times visible
- [ ] CTA is clear at the end
- [ ] Title card at start: "[Demo] — [What It Does]"
