# 💼 Main Offer

#flowops #strategy #offers

> Главная задача — зайти через быстрый результат, доказать ценность, перейти в долгосрочное партнёрство.

---

## Три главных оффера

### 1. [[Speed-to-Lead System]]
Для бизнесов, которые теряют клиентов из-за медленного ответа на заявки.
- **Цена:** $750–$2,500 setup + $300–$1,000/month поддержка
- **Боль:** получают заявки, но отвечают поздно

### 2. [[Ops Automation Sprint]]
Для команд, которые вручную переносят данные между инструментами.
- **Цена:** $1,000–$3,000 fixed
- **Боль:** часы рутины каждую неделю

### 3. [[AI Chatbot & Voice Agent]]
Для бизнесов с большим потоком входящих обращений.
- **Цена:** $1,500–$5,000 setup + $500–$2,000/month поддержка
- **Боль:** команда отвечает на одни и те же вопросы снова и снова

## Бесплатный вход — Free Automation Audit

**Цель:** начать диалог, не продавать в лоб.

**Формат:** Loom-видео / PDF / Notion-страница / 3 конкретные идеи автоматизации для бизнеса клиента.

**Когда использовать:** как opener в холодном аутриче и на Upwork.
