## What was done
- Replaced the earlier bilingual deck approach with two separate client-facing FlowOps decks:
  - English / USD version
  - Russian / KZT version
- Both decks use the same FlowOps visual system and the same core offer: **AI Sales Operating Systems**.
- Added commercial modeling and quantified business impact:
  - baseline example: `300 inbound leads / month`
  - modeled admin time saving: `40 hours / month`
  - modeled recovered leads: `15-30 / month`
  - modeled additional booked calls: `6-12 / month`
- Added commercial pricing:
  - EN: implementation `from $3,000`, monthly management `from $750/mo`, runtime stack `$120-300/mo`
  - RU: implementation `from 1.5 млн ₸`, monthly management `from 350 000 ₸/мес`, runtime `60-150 тыс. ₸/мес`

## Key findings
- Separate single-language decks read cleaner and feel more premium than one bilingual deck for outbound sales use.
- Numbers improve credibility when framed as an illustrative operating model rather than as a promise.
- The decks now sell a company-level system, not isolated automations or chatbot work.

## Blockers
- Pricing and modeled impact are strategic defaults, not a universal guarantee; they may need niche-specific variants later.

## Next steps
- Use the English deck for international prospects and the Russian deck for local / regional outreach.
- If useful, create niche-specific variants next: clinics, real estate, agencies, legal.
