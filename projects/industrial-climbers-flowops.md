- Без эмодзи в иконках. Все цвета — токены, никаких хардкодов в компонентах.
- A4 portrait, 794×1123px @ 96dpi для предсказуемого PDF-печать.

## Технический процесс
- HTML с инлайн-CSS → headless Chrome (`--print-to-pdf`) → A4 PDF.
- Команда печати:
  ```
  /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
    --headless=new --disable-gpu --no-pdf-header-footer --hide-scrollbars \
    --virtual-time-budget=2000 --print-to-pdf-no-header \
    --print-to-pdf=output.pdf file://path/to/index.html
  ```
- Стандартный паттерн переполнения в A4: при превышении контента низ страницы режется. Лечится сокращением списков, уменьшением `padding`/`line-height` карточек или фиксированной высотой плиток вместо `aspect-ratio`.
