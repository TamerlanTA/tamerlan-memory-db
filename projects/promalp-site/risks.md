## Открытые риски / блокеры
- **Нет контента от клиента.** Без фото бригады, POV-видео, кейсов «до/после», отзывов УК/КСК — сайт не запускается. Discovery-неделя из деки (слайд 13) ещё не проведена.
- **Не подтверждены реквизиты.** Юрстатус (ТОО/ИП), БИН, контактный номер для CTA, ссылка на Instagram бригады.
- **Не известны параметры lead-пайплайна.** Какой Airtable base / table ID использовать, какой Telegram chat ID для алертов бригадиру (Айдос), какой токен бота.
- **Матрица расчёта квиза не определена.** Нужны цены за погонный метр / м² по каждому типу работ × этажность × срочность. Без них квиз даёт фейковую цифру.
- **Домен не выбран.** Возможные варианты — `promalp.kz`, субдомен, или новый бренд. Влияет на SEO/брендинг с первого дня.
- **AI-обзвон (Vapi/Retell) — отдельный слой.** Сценарий передачи лида с сайта в обзвон должен быть синхронизирован: что сайт пишет в Airtable, что AI-агент читает.
- **Регуляторика РК.** Допуск ОТ, СОУТ, страхование ответственности — нужно подтвердить, что у клиента всё в порядке прежде, чем публиковать на сайте.
