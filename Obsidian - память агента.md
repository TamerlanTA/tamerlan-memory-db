4 слоя памяти:

1.  ﻿﻿﻿**Контекстное окно** - то что ты видишь прямо сейчас в чате. 200к токенов, живёт только в этой сессии. Когда закроешь - исчезнет.

2. ﻿﻿﻿**SQLite база данных** (/root/claudeclaw/store/claudeclaw.db) -  структурированная память между сессиями. Там хранятся checkpoint'ы - краткие саммари завершённых сессий с датой и темой.

3. ﻿**Obsidian vault** (~/obsidian-vault) - долгосрочная память в markdown-файлах. Сессионные заметки, исследования, контент, agent-memory.md с общей базой знаний агентов. Читаю и пишу через файловую систему

4. **Файлы конфигурации** - CLAUDE.md, агент-файлы в ~/.claude/agents/, скиллы в ~/.claude/skills/. Это не память" в классическом смысле но именно здесь живут правила роли и поведение - и они persist между сессиями постоянно.

### Для чего нужен

Когда ты закрываешь чат с Claude - он всё забывает. Следующий разговор начинается с нуля, как будто вы никогда не общались. Obsidian решает эту проблему: это приложение для заметок, где агент хранит всё что узнал. Открыл новый чат - агент читает свои записи и продолжает с того места где остановился

### Принцип работы

- ﻿﻿Vault (папка с заметками агента) хранится прямо на твоём компьютере - никакого облака
- ﻿﻿Файлы внутри - markdown (обычный текст с простым форматированием, открывается в любом редакторе)
- ﻿﻿Агент читает файлы через файловую систему (папки и файлы на компьютере) - так же как ты открываешь документ
- ﻿﻿Wikilinks (кликабельные ссылки между заметками вида название) связывают темы - агент переходит по ним как по веб-страницам
- ﻿﻿Структура: сессии / проекты / паттерны / ошибки / знания

### Инструкция как установить

1. ﻿﻿﻿Скачать Obsidian бесплатно: obsidian.mdc? - работает на Мас, Windows, телефоне
2. ﻿﻿﻿При первом запуске нажать "Create new vault" → выбрать любую папку на компьютере
3. ﻿﻿﻿В настройках Claude Code (файл CLAUDE.md - главный файл с инструкциями для агента) написать путь к vault и правило: "перед задачей читай, после - пиши"
4. ﻿﻿﻿Создать три папки внутри: "Сессии", "Проекты", "Знания"
5. ﻿﻿﻿Готово - агент сразу начнёт использовать как свой блокнот 

### Что показать на экране

Открыть Obsidian и показать: вот заметка из прошлой сессии → агент её читает сейчас → агент пишет новую. Граф связей (визуальная карта всех связанных заметок) между темами.
