# Production Readiness Checklist

Before shipping:

- [ ] Build passes.
- [ ] Tests pass.
- [ ] Typecheck/lint pass where applicable.
- [ ] Required env vars documented.
- [ ] Secrets are stored outside the repo.
- [ ] Database migrations reviewed.
- [ ] Rollback/recovery plan considered.
- [ ] Logs and monitoring adequate for first release.
- [ ] User-facing critical paths manually verified.
- [ ] Auth and permissions checked.
- [ ] Payment, messaging, outbound, or destructive actions have guardrails.
- [ ] Rate limits and external service failures handled.
- [ ] Support/debug instructions documented.
- [ ] Handoff summary written.
- [ ] Open risks accepted by owner or moved to next steps.
