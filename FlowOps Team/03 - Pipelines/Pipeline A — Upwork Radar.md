# 🎯 Pipeline A — Upwork Hot Lead Radar

#flowops #pipeline #upwork #acquisition

---

## Goal

> Find hot orders where clients are already ready to pay.
> **Target: 50–100 quality proposals per week** — not mass spam, but precise matches.

---

## Search Keywords

n8n · Make.com · Zapier · AI automation · AI agent · workflow automation · CRM automation · Airtable automation · ClickUp automation · Google Sheets automation · OpenAI API · Claude API · Vapi · Retell · chatbot · WhatsApp automation · Shopify automation

---

## How the Automation Should Work

1. Check new jobs every 15–30 minutes
2. Filter junk projects
3. Score each job by fit score
4. Identify best sales angle
5. Generate short proposal draft
6. Send top jobs to Telegram
7. Save everything to Airtable / Notion / Google Sheets

---

## Fit Score Criteria

- ✅ Budget $300+
- ✅ Specific business problem (not "I need something cool")
- ✅ Has CRM / API / webhooks / email / calendar
- ✅ Project resembles our case studies
- ✅ Few proposals submitted
- ✅ Payment verified client
- ✅ Normal client history
- ✅ Client not hunting for cheapest freelancer
- ✅ Potential for long-term support

---

## Output Fields

| Field | Description |
|-------|-------------|
| Job URL | Link to the posting |
| Budget | Stated or estimated |
| Client info | Rating, history, location |
| Pain summary | 1–2 sentence problem |
| Fit score | 1–10 |
| Recommended offer | Which of our 3 offers fits |
| Proposal draft | Ready to customize |
| Smart questions | 2 clarifying questions |
| Status | New / Applied / In Progress / Won / Lost |
| Follow-up date | When to check back |
