## Upwork Proposal

```
Hi [Name],

This is exactly the kind of workflow I build — connecting [tool 1], [tool 2], and AI so the process runs automatically instead of being handled manually.

1. [step 1]
2. [step 2]
3. [step 3]

A couple quick questions:
- [smart question 1]
- [smart question 2]

I can build a clean V1 first, then improve it based on real usage.
```
